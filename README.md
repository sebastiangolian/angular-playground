# angular-playground

Test app with angular

## Test app

[angular-playground](https://sebastiangolian.github.io/angular-playground)

## Getting started

```bash
npm install
```

```bash
npm run start
```

## Scripts

Build project

```bash
npm run build
```

Test project

```bash
npm run test
```

Lint project

```bash
npm run lint
```

Deploy files in gh-pages

```bash
npm run deploy
```
